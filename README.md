# LINE公式アカウント導入サービス - コーポレートウェブサイト

公式LINEアカウントを活用した集客・業務効率化サービスを提供する会社の公式ウェブサイトです。

## 📋 プロジェクト概要

このウェブサイトは、LINE公式アカウントの導入・運用代行・研修サービスを提供する企業向けに構築されたマルチページサイトです。視覚的で魅力的なデザインと、充実した機能を備えています。

## 🎯 主な目的

- **売上UP × コストダウン**を実現するLINEサービスの紹介
- 集客用・バックオフィス用・統合版・研修の4つのサービスライン展開
- 問い合わせフォームによるリード獲得
- チャット機能による即座の顧客サポート

## 📧 メール機能について

現在のフォーム送信機能はフロントエンドでシミュレートされています。実際の運用では以下の実装が必要です：

```javascript
// 実装例（要バックエンドAPI）
async function sendEmail(formData) {
    const response = await fetch('/api/contact', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
        },
        body: JSON.stringify(formData)
    });
    return response.json();
}
```
